Apurement des données

Le service de la géomatique et du registre foncier (SGRF) a mis en place un outil d’aide pour le traitement des erreurs, basé sur les fichiers d’erreurs publiés chaque semaine par l’OFS. Le fichier annexé filtre les données de l’OFS en utilisant les données de la mensuration officielle (MO) et oriente l’opérateur du RegBL sur la façon de traiter l’erreur.

Vous trouverez dans le fichier joint l’ensemble des erreurs organisées en 7 listes :

  • Liste 1: {{ NB_ERROR_LIST_1 }} Bâtiments existants dans le RegBL sans coordonnées => ajouter les coordonnées du bâtiment
  • Liste 2: {{ NB_ERROR_LIST_2 }} Coordonnées en dehors de la commune indiquée => corriger les coordonnées du bâtiment
  • Liste 3: {{ NB_ERROR_LIST_3 }} Divergence de NPA => corriger l'adresse ou contacter le SGRF pour modifier la géométrie du NPA
  • Liste 4: {{ NB_ERROR_LIST_4 }} Doublets d'adresses => corriger pour que les adresses soient univoques
  • Liste 5: {{ NB_ERROR_LIST_5 }} Définition du bâtiment => corriger selon les indications du SGRF (dernière colonne)
  • Liste 6: {{ NB_ERROR_LIST_6 }} Catégorie du bâtiment => corriger selon les indications du SGRF (dernière colonne)
  • Bâtiments manquants: {{ NB_ERROR_ISSUE_22 }} => renseigner le b'atiment dans le RegBL
Dans le cas où aucune erreur n’est constatée dans une liste, cette liste n’apparaît pas dans le fichier. Actuellement, par exemple, il n’y a aucune erreur dans les listes 1 à 3.

Le SGRF prévoit de vous transmettre ces listes tous les premiers mercredis du mois, afin que vous puissiez effectuer les corrections nécessaires directement dans le registre. Nous vous rappelons également que le document "Directives pour les communes sur l'harmonisation des données du RegBL et de la MO" contient des informations importantes sur le projet du RegBL.

Extension du RegBL

Nous avons le plaisir de vous informer que le projet d’extensions du RegBL touche à sa fin. Avec votre appui et en croisant les données de la MO et de l’établissement cantonal d’assurance et de prévention (ECAP), ce sont plus de 20'000 bâtiments sans usage d’habitation qui ont pu être enregistrés sur tout le canton entre fin 2021 et 2024, les constructions à vocation d’habitation ne pouvant être ajoutées que par les communes.

Les caractéristiques des constructions (GKAT, GKLAS) ont été remplies à l’aide des informations de la MO. Les périodes de construction des bâtiments ont, quant à elles, été renseignées sur la base des données de l’ECAP ou sur l’analyse des cartes historiques de la confédération.

Si vous constatez des erreurs dans les données, vous pouvez les corriger directement dans le registre.
